This is sample system info ...... where the details about system can be shown from browser.

Requirements:

1. Python 2.7 (Download and install python from python official website)
2. psutil python package ( after successfull installation of python..run command:  pip install psutil )
3. flask python package (pip install flask )

Download sysweb.py and modify the below lines accordingly.

HOST='localhost'  ( This can be ip of the interface or 0.0.0.0 for open to all interface or 127.0.0.1/'localhost' for loopback only )
PORT=2000 ( This is the port assigned with this )
DEBUG=False ( True or False )

Copy the sysweb.py to the path where to start and run it........
